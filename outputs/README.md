# System outputs

This directory defines deployable system identities.

Output names use lowercase fictional character names and are independent of the underlying hardware.

## Naming

Character namespaces are assigned based on the deployment role.

### Personal use

Personal workstations use character names from [Attack on Titan](https://en.wikipedia.org/wiki/Attack_on_Titan).

| Output   | Platform         | Hardware                | Purpose                |
| -------- | ---------------- | ----------------------- | ---------------------- |
| `eren`   | `aarch64-darwin` | `macbook-air-m4`        | Personal macOS system  |
| `mikasa` | `x86_64-linux`   | `asus-rog-strix-g614ji` | Personal NixOS desktop |

### Homelab

Homelab systems use character names from [Steins;Gate](https://en.wikipedia.org/wiki/Steins;Gate).

| Output  | Platform       | Hardware                | Purpose       |
| ------- | -------------- | ----------------------- | ------------- |
| `okabe` | `x86_64-linux` | `asus-rog-strix-g614ji` | NixOS homelab |
